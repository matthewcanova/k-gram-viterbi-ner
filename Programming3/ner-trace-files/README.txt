To help you assess the correctness of your code for each type of
feature, we are providing a large set of trace files for the ner
program. This folder contains the readable features, predictions, and
accuracy results that your program should produce using 8
different sets of arguments.

WORD:     using only the WORD feature
ABBR:     using only the WORD and ABBR features
CAP:      using only the WORD and CAP features
LOCATION: using only the WORD and LOCATION features
POS:      using only the WORD and POS features
POSCON:   using only the WORD and POSCON features
WORDCON:  using only the WORD and WORDCON features
ALL:      using all 7 of the features
